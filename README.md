# ①課題番号-プロダクト名
教科書出展図書検索アプリ

## ②課題内容（どんな作品か）
- Google Books APIを使い、小学校の国語の教科書に掲載されている図書を検索するアプリ
- ISBNを検索した後、カーリル（図書館検索）にジャンプして蔵書のステータスを確認できる
